## Features

This app was built with modern Android development practices and tools. It showcases the following concepts:

- [x] Kotlin
- [x] MVVM
- [x] Clean Architecture
- [x] Coroutines
- [x] Dagger Hilt
- [x] Retrofit
- [x] Moshi
- [x] Coil
- [x] Navigation Component
- [x] LiveData
- [x] ViewModel
- [x] Material Design
